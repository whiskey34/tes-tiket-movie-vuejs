## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
### to run tailwind CLI
```sh
npx tailwindcss -i ./src/assets/style.css -o ./src/output.css --watch
```

### Compile and Minify for Production

```sh
npm run build
```
